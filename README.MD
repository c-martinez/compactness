# Compactness measures

The compactness package contains Python implementation of various 2D and 3D compactness measures described in [2D and 3D Shape Descriptors](http://hdl.handle.net/10036/3026).

## Dependencies
 - Numpy

## Testing
Run:

```
$ nosetests
```
